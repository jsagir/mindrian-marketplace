# Mindrian Marketplace

Claude Code plugin marketplace for MindrianOS tools.

## Install

```bash
# Add the marketplace
claude plugin marketplace add jsagir/mindrian-marketplace

# Install MindrianOS
claude plugin install mindrian-os@mindrian-marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **mindrian-os** | Your AI innovation co-founder. 38 commands, 5 agents, knowledge graph integration. |
